Welcome to Task Manager

Task Manager is a simple, powerful web application to help you manage your daily tasks more efficiently. Whether you're juggling personal to-dos or project milestones, Task Manager helps you stay on top of everything.

Key Features

  • Organize Tasks: Sort tasks into Queued, In Progress, and Completed columns.
  • Live Editing: Update task title, description, and status with ease.
  • Interactive Interface: Click on a task to view or edit details instantly.
  • User Accounts: Secure login and user-specific task boards.

How It Works

Simply log in, add a new task, and watch your productivity flow. Drag between stages (coming soon), and manage everything from one intuitive dashboard.
